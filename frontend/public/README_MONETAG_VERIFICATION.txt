MONETAG VERIFICATION INSTRUCTIONS
=================================

When you download your Monetag verification file:

1. PLACE THE FILE HERE in this public folder
2. The file should be accessible at: https://www.muzikax.com/YOUR_FILE_NAME
3. Deploy your Next.js app to Vercel
4. Go back to Monetag dashboard and click "Verify" button

Common verification file names:
- monetag-verification-XXXXXXXXXX.html
- monetag-verification.txt
- Or similar filename provided by Monetag

After placing the file here, deploy using:
npm run deploy

Then verify on Monetag dashboard.
